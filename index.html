Introducción


Las estructuras de datos son muy importantes a la hora de ordenar datos o buscar por lo que es muy importante saber a qué se refiere las estructuras de datos en este sitio web encontraras la definición de las estructuras de datos, sus tipos, clasificación, etc. por lo que si te interesa este tema no dudes en seguir leyendo este artículo.

Se creo este sitio web de dar a conocer las estructuras de datos ya que en la programación es de suma importancia por lo que recabo alguna información y también se dejaron los recursos usados para mejorar la comprensión del lector por lo que no estaría demás visitar la página algoritmos y también ver el apartado de referencias en la cual se puede profundizar mejor con algunos libros que se utilizaron.

Se busca informar a todos los usuarios sobre las estructuras de datos por lo que se definieron algunos conceptos básicos sobre las estructuras por lo que si quieres revisar mas a fondo puedes pasar leer directamente desde el menú.
