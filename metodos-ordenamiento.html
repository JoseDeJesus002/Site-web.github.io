Metodos de ordenamiento

Ordenar significa permutar estos elementos de tal forma que queden de acuerdo con una distribución preestablecida.
En el procesamiento los métodos de ordenamiento se clasifican en dos grandes categorías:

  • Ordenación de arreglos.
  • Ordenación de archivos.

La primera se denomina ordenación interna, ya que los elementos o componentes de los arreglos se encuentran en la memoria principal de la computadora.

La segunda se denomina ordenación externa ya que los elementos se encuentran en archivos almacenados en dispositivos de almacenamiento secundarios, como discos, cintas, tambores, etcétera.

Ordenacion interna

La ordenación interna se puede aplicar en arreglos unidimensionales, estructuras de datos más avanzadas por lo que se puede aplicar en todos los tipos, para comenzar la ordenación se divide en dos tipos:

  • Métodos directos.
  • Métodos logarítmicos.

Los métodos directos poseen una característica de su implementación es relativamente sencilla y son fáciles de comprender, aunque se puede complicar cuando se tiene un numero indeterminado de registros, mientras que los métodos logarítmicos son mas complejos, sin embargo, son más eficientes ya que no necesitan recorrer todo y tampoco necesitan comparar varios datos.

    Los métodos directos más conocidos son:
  • Ordenación por intercambio.
  • Ordenación por inserción.
  • Ordenación por selección.

Ordenación por inserción directa

Este método de ordenación es el que se utiliza generalmente en los jugadore de cartas cuando ordenan, de ahí también que se conozca como el método de la baraja.

La idea central de este algoritmo consiste en insertar un elemento del arreglo en su parte izquierda que ya se encuentra ordenada y esto se repite hasta n veces.

int p, j;
int aux;
for (p = 1; p < A.length; p++){ // desde el segundo elemento hasta
aux = A[p]; // el final, guardamos el elemento y
j = p - 1; // empezamos a comprobar con el anterior
while ((j >= 0) && (aux < A[j])){ // mientras queden posiciones y el
// valor de aux sea menor que los
A[j + 1] = A[j]; // de la izquierda, se desplaza a j--; // la derecha
}
A[j + 1] = aux; // colocamos aux en su sitio
}

Ordenación por selección directa

Este es uno de los métodos mas eficientes, esto por que su comportamiento es mejor que los otros mencionados y su programación es fácil y comprensible, este método no se recomienda utilizarlo en un arreglo cuando es mediano o grande.

La idea de este algoritmo consiste en buscar el segundo elemento mas pequeño del arreglo y se coloca en segunda posición, luego se busca el segundo elemento mas pequeño del arreglo y se coloca en la segunda posición y así continua el proceso hasta que queda totalmente ordenado.
